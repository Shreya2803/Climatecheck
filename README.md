A simple, interactive weather web application built using HTML, CSS, and JavaScript. This app lets users check the current weather conditions in any city by entering the city's name. It fetches real-time data from the OpenWeather API and displays information such as temperature, weather conditions, humidity, and wind speed.

## Features

- **City-based Weather Search**: Enter any city name to get its real-time weather information.
- **Live Weather Data**: Displays temperature, humidity, wind speed, and an icon representing the weather condition.
- **User-friendly Interface**: Clean, responsive design with clear, accessible information.
- **Error Handling**: Displays an error message for invalid city names.
